# <a name="deployment-advisors-for-office-365-services"></a>Conseillers de déploiement pour les services Office 365

Les conseillers de déploiement pour Office 365 vous donnent une assistance auto-guidée pour la configuration d'Office 365 pour votre organisation. Il vous suffit de sélectionner un guide de configuration ou un conseiller et de vous connecter à Office 365. 

Lorsque vous sélectionnez les fonctionnalités et les options que vous souhaitez déployer, le conseiller élabore un plan d'installation détaillé adapté à vos besoins. Vous obtiendrez un ensemble complet d'instructions, de vidéos, d'Articles de référence et de scripts. Certains conseillers ont une automatisation qui modifiera certains de vos paramètres, tandis que dans d'autres zones du conseiller, vous modifiez vos paramètres et vos données avec le conseiller pour vous aider. Vous pouvez utiliser ces conseillers à tout moment, même lors de la planification ou après avoir configuré les services, pour en savoir plus sur les fonctionnalités et les options d'Office 365.
  
## <a name="windows-10-with-office-365"></a>Windows 10 avec Office 365

[Conseiller de déploiement Microsoft 365](https://aka.ms/microsoft365setupguide)
  
Microsoft 365 est une solution complète et intelligente qui inclut Office 365, Windows 10 et Enterprise Mobility + Security. Microsoft 365 permet à tout le monde d'être créatif et de collaborer en toute sécurité. Utilisez ce conseiller pour configurer des appareils Windows 10 ou mettre à niveau les ordinateurs des utilisateurs vers Windows 10, avec un déploiement facultatif d'applications Office, Windows Analytics et Windows Defender Advanced Threat Protection (Microsoft 365 Enterprise E5 plan only).


## <a name="mail-migration-and-protection"></a>Protection et migration de messagerie

### <a name="prepare-your-environment"></a>Préparer votre environnement 
Le [Guide préparer votre environnement](https://go.microsoft.com/fwlink/?linkid=2005213) est le point de départ. Faites-nous part de vos objectifs de déploiement pour vous assurer que vous ajoutez des domaines, créez des utilisateurs et attribuez des licences dans le bon ordre. Ceci est particulièrement important si vous envisagez de migrer le courrier électronique ou de configurer un déploiement hybride. 

### <a name="exchange-migration-advisor"></a>Conseil sur la migration Exchange
[Office 365 mail migration Advisor](https://aka.ms/office365setup) vous aidera à déplacer des boîtes aux lettres à partir de votre système de messagerie actuel vers Exchange Online dans Office 365 avec des outils automatisés et des conseils détaillés. Nous vous recommandons de choisir le meilleur chemin de migration pour votre organisation en fonction de votre système de messagerie actuel, le nombre de boîtes aux lettres que vous souhaitez migrer et la manière dont vous prévoyez de gérer les utilisateurs et l'accès des utilisateurs.
  
L' [Assistant contacts et calendrier gmail](https://aka.ms/gmailcontactscalendar) fournit des instructions pas à pas pour migrer vos contacts gmail et vos éléments de calendrier Google vers Office 365. Lorsque vous migrez la boîte aux lettres gmail d'un utilisateur vers Office 365, les messages électroniques sont migrés, mais pas les contacts et les éléments de calendrier. Ce guide décrit les étapes à suivre pour importer des contacts gmail et des éléments de calendrier Google vers Office 365.
  
### <a name="exchange-online-protection"></a>Exchange Online Protection
Microsoft [Exchange Online Protection (EoP)](https://aka.ms/EOPguidance) est un service de filtrage du courrier basé sur le Cloud qui permet de protéger votre organisation contre le courrier indésirable et les programmes malveillants et inclut des fonctionnalités permettant de protéger votre organisation des violations de stratégie de messagerie.

### <a name="sharepoint-online"></a>SharePoint Online
  
Le [Guide de démarrage rapide de SharePoint Online](https://aka.ms/SPOquickstartguide) vous montre comment configurer SharePoint dans le Cloud pour le stockage de documents et la gestion de contenu. Cette configuration de base est tout ce dont vous avez besoin si vous ne disposez pas d'une quantité importante de données à migrer ou d'un serveur SharePoint Server sur site que vous souhaitez continuer à exécuter dans une configuration hybride.
  
Le [conseiller de déploiement SharePoint Online](https://aka.ms/spoguidance) vous aide à choisir l'option de déploiement qui convient le mieux à votre organisation et vous explique comment configurer les fonctionnalités de SharePoint Online pour répondre aux besoins de votre entreprise. Choisir entre un déploiement sur le Cloud, un déploiement hybride ou une migration locale vers le Cloud. Ensuite, suivez les étapes de configuration des fonctionnalités SharePoint Online, telles que le stockage et le partage de fichiers, le partage de fichiers externe, les collections de sites, les paramètres globaux et les profils utilisateur, ainsi que votre site d'équipe Office 365.
  
## <a name="security-and-identity"></a>Sécurité et identité

### <a name="azure-active-directory-connect-azure-ad-connect-advisor"></a>Azure Active Directory Connect (Azure AD Connect) Advisor
Le [conseiller Azure ad Connect](https://aka.ms/aadconnectpwsync) est un guide pas à pas qui vous montre comment ajouter des informations de compte d'utilisateur à Office 365 sans créer manuellement chaque utilisateur. Il explique également comment configurer la synchronisation de hachage de mot de passe ou l'authentification directe, afin que les utilisateurs puissent se connecter à leur messagerie et à votre domaine à l'aide du même mot de passe. Si vous choisissez de configurer la connexion fédérée avec AD FS, ce guide vous explique également comment déployer AD FS sur un nouveau serveur ou dans une batterie de serveurs Windows Server 2012 R2 existante. Azure AD Connect remplace les anciennes versions des outils d'intégration des identités, tels que dirSync et Azure AD Sync, et sert principalement à ajouter des utilisateurs et d'autres données à Azure Active Directory pour Office 365.
  
### <a name="azure"></a>Azure
Le [Guide de configuration d'Azure ad basique](https://aka.ms/azureadbasic) vous aide à configurer les fonctionnalités, telles que la gestion des accès basée sur un groupe, la réinitialisation du mot de passe en libre-service pour les applications Cloud et le proxy d'application Azure Active Directory pour la publication d'applications Web locales.
  
Le [Guide de configuration d'Azure ad Premium](https://aka.ms/aadpguidance) vous permet d'activer les nombreuses fonctionnalités d'Azure Active Directory Premium (Azure ad Premium), qui fournit des fonctionnalités de gestion des identités telles que l'authentification multifacteur, l'authentification unique (SSO), l'inscription de l'appareil gestion des mots de passe et des groupes, et surveillance de la sécurité.
  
### <a name="verify-your-domain"></a>Verify your domain
Le conseiller [vérifier votre domaine dans office 365](https://aka.ms/verifyyourdomaino365) Advisor vous aide à personnaliser Office 365 en ajoutant votre propre nom de domaine (également appelé nom de votre site Web).
  
## <a name="communication-and-online-conferencing"></a>Communication et conférence en ligne

### <a name="office-365-groups"></a>Groupes Office 365
[Guide de configuration des groupes Office 365](https://aka.ms/groupsguide)

Les groupes Office 365 sont un espace de travail partagé pour les messages électroniques, les conversations, les fichiers et les événements où les membres du groupe peuvent effectuer des opérations collectives. Les groupes dans Office 365 vous permettent de choisir un ensemble de personnes avec lesquelles vous souhaitez collaborer et de configurer facilement une collection de ressources à partager pour ces personnes. Vous n'avez pas à vous soucier de l'affectation manuelle d'autorisations à toutes ces ressources, car l'ajout de membres au groupe leur accorde automatiquement les autorisations dont ils ont besoin pour les outils fournis par votre groupe.
  
### <a name="microsoft-teams"></a>Microsoft Teams

[Conseiller en matière de déploiement de teams](https://aka.ms/teamsguidance)
  
Microsoft teams est le service de collaboration basé sur la conversation dans Office 365 qui fournit des espaces de travail d'équipe pour les conversations, les appels, les réunions et les messages privés. Le conseiller de déploiement de teams vous permet de configurer et de configurer l'expérience de teams optimale avec Exchange Online, SharePoint Online, OneDrive entreprise et les groupes Office 365. Elle comprend des informations sur la configuration des paramètres de client, des équipes et des canaux pour vos projets, des appels et des réunions, la messagerie, des onglets pour accéder rapidement aux applications, les robots d'information et les connecteurs vers des services tiers.
  
### <a name="skype-for-business"></a>Skype Entreprise

[Guide de démarrage rapide de Skype entreprise](https://aka.ms/SfBquickstartguide)
  
Familiarisez-vous avec Skype entreprise pour la messagerie instantanée, la présence, les réunions en ligne et le partage d'écran. Si vous n'avez pas besoin de fonctionnalités avancées, cette configuration de base vous suffit.
  
[Conseiller de déploiement Skype entreprise](https://aka.ms/skypeguidance)
  
Le conseiller de déploiement Skype entreprise vous permet de vous familiariser avec Skype entreprise online. Ce conseiller vous montre comment configurer les fonctionnalités de Skype entreprise, comme la messagerie instantanée, les réunions en ligne, la vidéoconférence, la Conférence sur le réseau téléphonique commuté (RTC), le cloud privé (PBX) et la diffusion de réunion Skype. Des étapes sont fournies pour la configuration d'un environnement hybride, qui vous permet de connecter Skype entreprise Online à votre déploiement Skype entreprise local.

[Introduction à Yammer: votre réseau social d'entreprise](https://aka.ms/yamquickstartguide)
  
Ce guide de démarrage rapide vous montre comment déployer Yammer dans votre organisation. Si vous ne disposez pas de réseaux Yammer existants pour consolider ou migrer, cette configuration de base vous suffit.
  
[Guide de configuration Yammer Enterprise](https://aka.ms/yammerdeploy)
  
Le conseiller de déploiement Yammer Enterprise vous aide à faire fonctionner votre organisation avec Yammer Enterprise.
