# <a name="move-in-ready"></a>Solutions prêtes à installer

 **Résumé :** Obtenez les détails sur l'ensemble d'options de stockage cloud qui sont intégrées dans les offres cloud de Microsoft.
  
Les solutions de stockage « prêtes à installer » :
  
- utilisent des solutions prépackagées immédiatement ;
    
- sont prêtes à l’emploi, intégrées aux services existants ;
    
- fournissent une personnalisation limitée.
    
Les sections suivantes décrivent les détails de chaque option de stockage « prête à installer ».
  
## <a name="sharepoint-online-in-office-365"></a>SharePoint Online dans Office 365

### <a name="features"></a>Fonctionnalités

- Les fichiers sont sécurisés, fiables et accessibles à partir de plusieurs appareils et de partout
    
- Le contenu peut être facilement découvert via une recherche
    
- Des flux de travail et stratégies de rétention facilitent l’utilisation et la gestion du contenu
    
- Le contenu est accessible via des API web, un navigateur ou hors connexion
    
### <a name="common-uses"></a>Utilisations courantes

- Portail d’entreprise ou d’organisation
    
- Collaboration en équipe
    
- Recherche d’entreprises
    
### <a name="key-storage-scenarios"></a>Principaux scénarios de stockage

- Collaborer
    
- Communications d’entreprise
    
- Gérer des vidéos
    
### <a name="resources"></a>Ressources

Pour plus d'informations, cliquez [ici]((https://support.office.com/article/SharePoint-Online-and-OneDrive-for-Business-software-boundaries-and-limits-8f34ff47-b749-408b-abc0-b605e1f6d498)).
  
Pour plus d'informations de coût, cliquez [ici]((https://products.office.com/business/compare-office-365-for-business-plans)).

## <a name="onedrive-for-business"></a>OneDrive Entreprise

### <a name="features"></a>Fonctionnalités

- Le stockage est sécurisé, fiable et accessible à partir de plusieurs appareils et de partout
    
- L’utilisateur peut facilement découvrir le contenu via une recherche
    
- Le contenu est accessible via des API web, un navigateur ou hors connexion
    
- Chaque utilisateur reçoit 1 To de stockage
    
### <a name="common-uses"></a>Utilisations courantes

- Banque de documents des utilisateurs individuels
    
- Collaboration ad hoc sur des documents
    
### <a name="key-storage-scenarios"></a>Principaux scénarios de stockage

- Sauvegarder des fichiers

## <a name="visual-studio-team-services"></a>Visual Studio Team Services

### <a name="features"></a>Fonctionnalités

- Créer des journaux des travaux en souffrance, travailler sur des sprints et suivre les modifications et les bogues

## <a name="microsoft-intune"></a>Microsoft Intune

### <a name="features"></a>Fonctionnalités

- Tous les logiciels que vous déployez sont empaquetés et chargés sur le stockage cloud Microsoft Intune.
    
- L’abonnement comprend 20 Go de stockage avec la possibilité d’acheter de l’espace supplémentaire, différents logiciels et rapports d’inventaire de matériel informatique, la gestion des stratégies, la publication d’applications mobiles, des alertes, des analyses et 99,99 % de temps d’activité garanti par contrat de niveau de service (SLA).
    
### <a name="common-uses"></a>Utilisations courantes

- Déploiement de logiciels

## <a name="azure-site-recovery"></a>Azure Site Recovery

### <a name="features"></a>Fonctionnalités

- Protection, réplication et récupération simples et automatisées sur Azure
    
- Analyse du fonctionnement et récupération orchestrée en continu
    
- Prend en charge les disques gérés et les ordinateurs virtuels Ubuntu
    
- Prends en charge Windows Server 2016 et des espaces de stockage
    
### <a name="common-uses"></a>Utilisations courantes

- Des machines virtuelles s’exécutant dans un environnement utilisant Hyper-V peuvent être répliquées entre deux centres de données ou sur Azure
    
- InMage Scout permet la protection de VMware et d’environnements physiques
    
### <a name="key-storage-scenarios"></a>Principaux scénarios de stockage

- Sauvegarder des fichiers
    
### <a name="resources"></a>Ressources

Pour plus d'informations, cliquez [ici](http://azure.microsoft.com/services/site-recovery/?WT.srch=1&amp;WT.mc_ID=kn0iWCHA).

## <a name="azure-backup"></a>Sauvegarde Azure

### <a name="features"></a>Fonctionnalités

- Protège avec fluidité les données reçues de Windows Server et de Windows Server Essentials
    
- Inclut Microsoft System Center Data Protection Manager
    
- Les données sont stockées dans un stockage géorépliqué qui conserve six copies de vos données sur deux centres de données Azure
    
- Prend en charge les modifications delta, une fréquence définie, la compression, le chiffrement, une conservation plus longue et la limitation de bande passante
    
### <a name="common-uses"></a>Utilisations courantes

- Protéger les données provenant de diverses sources locales (SharePoint, SQL, Exchange) et Azure (machines virtuelles)
    
### <a name="key-storage-scenarios"></a>Principaux scénarios de stockage

- Sauvegarder des fichiers
    
### <a name="resources"></a>Ressources

Pour plus d'informations, cliquez [ici]((http://azure.microsoft.com/services/backup/)).
  
Pour plus d'informations sur les coûts, cliquez [ici]((http://azure.microsoft.com/pricing/details/backup/)).
