## <a name="call-the-cmdlet-directly-with-invoke-command"></a><span data-ttu-id="1ea34-156">Appel de la cmdlet directement avec la commande Invoke</span><span class="sxs-lookup"><span data-stu-id="1ea34-156">Call the cmdlet directly with Invoke-Command</span></span>

<span data-ttu-id="1ea34-p113">L’importation d’une session PowerShell distante (étape 3) peut être un processus long, car _toutes_ les cmdlets Exchange Online sont importées. En guise d’alternative à l’utilisation d’**Import-PSSession**, vous pouvez appeler les cmdlets que vous voulez utiliser directement avec **Invoke-Command**. As an alternative to using **Import-PSSession**, you can call cmdlets you want to use directly with **Invoke-Command**. For example, to call the **Get-Milbox** cmdlet, substitute this syntax for the `Import-PSSession $Session` command in Step 3:</span></span>
  
```
Invoke-Command -Session $Session -ScriptBlock {Get-Mailbox}
```
