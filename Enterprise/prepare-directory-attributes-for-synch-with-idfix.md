1. <span data-ttu-id="e2f82-110">Avant de commencer à apporter des modifications, examinons les recommandations présentées par IdFix.</span><span class="sxs-lookup"><span data-stu-id="e2f82-110">Before you start making changes, take a look at the recommendations presented by IdFix.</span></span>

Si plusieurs erreurs sont associées à un attribut unique, les erreurs sont combinées sur une seule ligne. Si possible, IdFix présente une recommandation pour un correctif dans la colonne **METTRE À JOUR**. Le correctif est basé sur une vérification des autres attributs associés à un objet. 3. <span data-ttu-id="e2f82-p103">Si IdFix a une suggestion pour un correctif pour une erreur de duplication, le correctif est identifié par un des trois indicateurs au début de la valeur dans la colonne de la **mise à jour** , par exemple, `[E]john.doe@contoso.com`.  - <span data-ttu-id="e2f82-p104">**[C]** Action suggérée **TERMINER**. La valeur n'a pas besoin d'être modifiée.</span><span class="sxs-lookup"><span data-stu-id="e2f82-p104">**[C]** Suggested action **COMPLETE**. The value does not need to be edited.</span></span>

4. <span data-ttu-id="e2f82-p107">Une fois que vous avez lu et compris les erreurs, mettre à jour de l’entrée dans la colonne **mettre à jour** avec vos modifications, puis dans l' **ACTION** colonne Sélectionnez ce que vous voulez IdFix à suivre pour implémenter la modification. Pour résoudre ce problème, sélectionnez la colonne **ACTION** **complète** pour l’utilisateur avec la valeur correcte et marquer la colonne **ACTION** **Supprimer** l’autre utilisateur. To fix this, mark the **ACTION** column **COMPLETE** for the user with the correct value, and mark the **ACTION** column **REMOVE** for the other user.
