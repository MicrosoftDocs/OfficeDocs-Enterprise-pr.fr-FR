#### <a name="how-do-i-upgrade"></a><span data-ttu-id="229a3-195">Comment passer ?</span><span class="sxs-lookup"><span data-stu-id="229a3-195">How do I upgrade?</span></span>

<span data-ttu-id="229a3-196">Pendant que vous trouverez plus d’informations sur la mise à niveau dans l’ensemble de contenu de [mise à niveau vers Project Server 2010](https://go.microsoft.com/fwlink/p/?linkid=841812) , il est important de comprendre qu’il existe deux méthodes distinctes, que vous pouvez utiliser pour mettre à niveau :</span><span class="sxs-lookup"><span data-stu-id="229a3-196">While details on how to upgrade can be found in the [Upgrade to Project Server 2010](https://go.microsoft.com/fwlink/p/?linkid=841812) content set, it is important to understand that there are two distinct methods you can use to upgrade:</span></span> 
  
- <span data-ttu-id="229a3-p114">**Mise à niveau par attachement de base de données :** Cette méthode met à jour uniquement le contenu de votre environnement et pas les paramètres de configuration. Il est nécessaire si vous mettez à niveau à partir d’Office Project Server 2007 déployé sur du matériel qui prend uniquement en charge un système d’exploitation 32 bits. Il existe deux types de méthodes de mise à niveau par attachement de base de données :</span><span class="sxs-lookup"><span data-stu-id="229a3-p114">**Database-attach upgrade:** This method only upgrades the content for your environment, and not the configuration settings. There are two types of database-attach upgrade methods:</span></span> 
    
  - <span data-ttu-id="229a3-200">**Mise à niveau complète par attachement de base de données** - migre les données de projet stockées dans les bases de données Office Project Server 2007, ainsi que les données de site Microsoft Project Web App (PWA) stockées dans une base de données de contenu SharePoint.</span><span class="sxs-lookup"><span data-stu-id="229a3-200">**Database-attach Full upgrade** - Migrates the project data stored in the Office Project Server 2007 databases, plus the Microsoft Project Web App (PWA) site data stored in a SharePoint content database.</span></span> 
    
  - <span data-ttu-id="229a3-201">**Mise à niveau principale par attachement de base de données** : migre uniquement les données de projet stockées dans les bases de données Project Server.</span><span class="sxs-lookup"><span data-stu-id="229a3-201">**Database-attach Core upgrade** - Migrates only the project data stored in the Project Server databases.</span></span> 
    
- <span data-ttu-id="229a3-p115">**Mise à niveau sur Place**: les données de configuration pour la batterie de serveurs et tous les contenus de la batterie de serveurs sont mise à niveau sur le matériel existant, selon un ordre défini. Lorsque vous démarrez le processus de mise à niveau sur place, le programme d’installation prend toute la batterie en mode hors connexion et les sites Web et les sites de Microsoft Project Web App ne sont pas disponibles tant que la mise à niveau est terminée et le programme d’installation redémarre le serveur. Après avoir commencé à une mise à niveau sur place, vous ne pouvez pas suspendre la mise à niveau ou revenir à la version précédente. Il est fortement recommandé pour rendre une image en miroir de votre environnement de production et effectuez la mise à niveau sur place pour cet environnement et non votre environnement de production.</span><span class="sxs-lookup"><span data-stu-id="229a3-p115">**In-Place upgrade**: The configuration data for the farm and all content on the farm is upgraded on the existing hardware, in a fixed order. When you start the in-place upgrade process, setup takes the entire farm offline and the Web sites and Microsoft Project Web App sites are unavailable until the upgrade is finished, and then setup restarts the server. After you begin an in-place upgrade, you cannot pause the upgrade or roll back to the previous version. It is highly suggested to make a mirrored image of your production environment and to do the in-place upgrade to this environment, and not your production environment.</span></span>
