## <a name="office-365-directory-synchronization"></a><span data-ttu-id="1a922-107">Synchronisation d’annuaires Office 365</span><span class="sxs-lookup"><span data-stu-id="1a922-107">Office 365 directory synchronization</span></span>

<span data-ttu-id="1a922-p102">Vous pouvez utiliser soit synchronisée identité ou les identités fédérées entre votre organisation locale et d’Office 365. C’est le scénario de synchronisation de répertoire les plus courants. This is the most common directory synchronization scenario. Pass-through authentication or Federated identity, allows you to manage your users on-premises and they are authenticated by your on-premises directory. Federated identity requires additional configuration and enables your users to only sign in once. For details, read [Understanding Office 365 Identity and Azure Active Directory](about-office-365-identity.md).</span></span>
