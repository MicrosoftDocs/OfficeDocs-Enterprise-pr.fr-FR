<span data-ttu-id="a5dcb-121">Les colonnes de données affichées sont :</span><span class="sxs-lookup"><span data-stu-id="a5dcb-121">Data columns shown are:</span></span>

- <span data-ttu-id="a5dcb-p105">**ID** : numéro d’identification de la ligne, également connu sous forme d’un ensemble de points de terminaison. This ID is the same as is returned by the web service for the endpoint set.</span></span>

- <span data-ttu-id="a5dcb-p106">**Catégorie** : indique si l’ensemble de points de terminaison est associé à la classe « Optimiser », « Autoriser » ou « Par défaut ». This column also lists which endpoint sets are required to have network connectivity. For endpoint sets which are not required to have network connectivity, we provide notes in this field to indicate what functionality would be missing if the endpoint set is blocked. - <span data-ttu-id="a5dcb-p107">**ER** : cette option est définie sur **Oui** si l’ensemble de points de terminaison est pris en charge sur Azure ExpressRoute avec des préfixes d’itinéraire Office 365. - <span data-ttu-id="a5dcb-p108">**Adresses** : répertorie les noms de domaine complets ou noms de domaines génériques et plages d’adresses IP pour l’ensemble de points de terminaison.
